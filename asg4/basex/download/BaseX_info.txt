BaseX: XPath/Xquery 8.6.7,  Platform-independent, Java 7 is required

Steps:
 - Uncompress basex.zip to C:\basex
 - execute C:\basex\download\BaseX867.exe
 - run "Basex GUI" from desktop or StartMenu
    Options -> Preferences... ->
      Database Path: C:/basex/data
      Repository Path: C:/basex/queries
